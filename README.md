Installation
----------

- Install the latest [Node.js](https://nodejs.org/download/) (which includes the NPM package manager) .
- Checkout the project and install the NPM dependencies

```bash
git clone https://github.com/neeedo/neeedo-sails.git
npm install
```

Run the project
----------

```bash
npm start
```
